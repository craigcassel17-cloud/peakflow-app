# PeakFlow Business App

A mobile-friendly React + Vite app prototype for PeakFlow Exterior Cleaning of Georgia.

## What is included
- Dashboard
- CRM / lead tracking
- Quote builder
- Schedule tab
- Payments tab
- Field notes / photo placeholder

## How to run locally
1. Install Node.js
2. Open a terminal in this folder
3. Run:
   npm install
   npm run dev

## How to deploy on Vercel
1. Create a GitHub account if needed
2. Create a new repo and upload these files
3. Log into Vercel
4. Import the GitHub repo
5. Click Deploy

## Important
This is a front-end prototype.
It does not yet include:
- database/cloud sync
- Square API
- text messaging integration
- real image storage
